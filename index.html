Owl - KeePass 2.x extension for the browser

Like its predecessor, Tusk, Owl is a Chrome and Firefox extension that adds autofill functionality for KeePass 2.x databases. Owl is open source and you can view and download the source code on GitHub.

Owl aims to finalize a lot of features that were originally planned to be in Tusk 2.0. Among them is finalized WebDAV support as well as the ability to add and edit password entries directly in the extension. With Owl, you'll no longer have to rely on other tools to manage your password storage in a safe and secure way.

Security

Owl builds on the same security concepts as Tusk, expanding where necessary. It is not affected by the bug where master passwords are stored on the disk. Owl uses KdbxWeb for crypto operations.

Privacy Policy

  • Owl may transmit and receive user data between the user's own storage at one of the cloud storage providers (e.g. Google Drive, Dropbox) and the Owl extension. When doing so, data is always encrypted using the mechanism supported by the cloud provider.
  • Owl does not transmit data to anywhere not mentioned above.
  • Owl only scans webpages when the user has opted-in to fill passwords on that particular website.
  • When Owl scans a webpage, it is for the sole purpose of identifying username and password fields, so that they may be automatically filled at the user's request.
